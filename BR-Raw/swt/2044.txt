[Presentations] AIOBE on CTabFolder setSelectionBackground
I20070503-1400
We get an AIOBE when starting Eclipse of a platform where the display has a bit depth of 8 (specifically AIX in our case).
The problem seems to stem from the following lines in CTabFolder.setSelectionBackground :
if (getDisplay().getDepth()
< 15) {
// Don't use gradients on low color displays
colors = new Color[] {colors[colorsLength
- 1]};
percents = new int[] {};
}
Causing this to be true, will trigger the exception every time.
