I-Beam displayed in wrong control (Run... dialog)
1) Open the Run dialog (i.e. click on down arrow beside the running man, select "Run..." from pop
up)
2) Select any item in the "Java Application" category (create one if needed)
3) click on "Arguments" tab.
4) click in "VM Arguments:" text field.
5) type characters.
notice that, the i-beam appears and disappears in the "Configurations:" area once for each
character that is typed.
