CTabFolder with a disposed top right control cannot be rendered
When a composite's child gets disposed there aren't any layout problems.
Shouldn't the same apply to CTabFolder?
Display display = new Display();
Shell shell = new Shell(display, SWT.SHELL_TRIM);
shell.setLayout(new FillLayout());
CTabFolder folder = new CTabFolder(shell, SWT.CLOSE);
Composite c = new Composite(folder, SWT.NONE);
folder.setTopRight(c);
c.dispose();
shell.open();
while (!shell.isDisposed()) {
if (!display.readAndDispatch())
display.sleep();
}
display.dispose();
