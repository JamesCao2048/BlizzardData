Create a MessageBox with SWT.SHEET style gives 2 apple menus
Get the latest org.eclipse.swt for Cocoa, and org.eclipse.swt.examples from HEAD.  (20090421)
Run ControlExample.
Go to the Dialog tab.
Select Dialog Type: MessageBox.
Select SWT.SHEET style.
Click "Create Dialog".
There will be 2 Apple mens in the menu bar.
