Set/Get API of DateTime widget crashes the control example
Build ID: I20080207-1530
Steps To Reproduce:
1. open control example
2. select DateTime tab
3. Focuses the date, time or calendar widget by clicking on it.
4. Make some changes to the widget's numbers (either keyboard or mouse input).
5. Click the Set/Get API button in Other section.
More information:
The program crashes when above actions take place.
