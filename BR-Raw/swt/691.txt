File open dialog doesn't remember last opened folder.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.1.3) Gecko/20090824 Firefox/3.5.3
Build Identifier: Snow Leopard 10.6.x
It remembered last open directory on Tiger/Leopard 10.5 but doesn't in Snow Leopard.
Reproducible: Always
Steps to Reproduce:
1. Open File Dialog and choose some file in some specific directory.
2. Open it again but the last directory will not be shown.
