In ScrolledComposite, vertical scrollbar does not appear
"Jay Wootton" wrote in eclipse.platform.swt newsgroup:
> I'm using ScrolledComposite but my vertical scroll bar never appears.
In
debugging the ScrolledComposite class I think I see the problem.
In the
needVScroll() method, the data member 'expandHorizontal' is checked.
I
believe needVScroll() should be checking the 'expandVertical' data member.
To
get around this problem I must call setExpandHorizontal( false ), then my
vertical scroll bar appears.
But I need to call setExpandHorizontal( true )
because I need the content to occupy the entire width.
