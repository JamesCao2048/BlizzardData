Show some XML attributes in Ant editor outline view
Ant's editor allows to edit XML files and have a good capability to show outline of xml document. It will be very convenient to see values of some attributes in the outline view of Ant editor in the same node as XML element, i.e. "name" and "id". It will aloow to easily navigate through the groups of the repeatable elements. List of such attributes could be configurable in the editor properties.
